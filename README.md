## Getting Started

### Dependencies

- Vite
- React
- SCSS
- ESLint
- Prettier
- Husky

### Installing

- Clone the repository

```
git clone git@github.com:bmartin97/quick-grammar.git
```

- Open the project directory

```
cd ./quick-grammar
```

- Install dependencies

```
npm install
```

- Run development mode

```
npm run dev
```
